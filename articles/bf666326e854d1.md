## はじめに

Unity CLI Loopは、Claude CodeなどのAIエージェントからUnity Editorを操作できるようにするツールです。コンパイルやPlayモード制御、スクリーンショット撮影、UI操作シミュレーションなど、通常はGUIで行う操作をAIエージェント経由で実行できるようになります。

以前、前身のuLoopMCPを使ってAI駆動でUnityゲーム開発を試す記事を書きました。

https://zenn.dev/unsoluble_sugar/articles/cd8d59be7b8f85

その後「Unity CLI Loop」にリネームされ、MCPではなくCLIツールがメインとなっています。本記事はClaude Codeユーザー向けに、名称変更後の導入手順をまとめたものです。

https://github.com/hatayama/unity-cli-loop

## 環境

- OS: macOS Sequoia 15.7.4
- Unity 6000.3.10f1

## パッケージのインストール

まずはUnity側にパッケージをインストールします。Unity Package Managerからgit URLを指定してインストールします。

`Window` > `Package Management` > `Package Manager` を開き「+」ボタンから「Install package from git URL」を選択します。

![Package Manager](https://storage.googleapis.com/zenn-user-upload/e88284c8176b-20260324.png)

以下のURLを入力して「Install」ボタンをクリックします。

```
https://github.com/hatayama/unity-cli-loop.git?path=/Packages/src
```
![git URL入力](https://storage.googleapis.com/zenn-user-upload/6d7d591a1b95-20260324.png)


## CLIツールのインストール

次に、AIエージェントとUnity Editorを橋渡しするCLIツール（`uloop`コマンド）をインストールします。

`Window` > `Unity CLI Loop` > `Settings` を開き「Install CLI」ボタンをクリック。ボタンが「Up to date」に変わればインストール完了です。

![CLI Settings](https://storage.googleapis.com/zenn-user-upload/bcb0c034c0fb-20260324.png)

ターミナルでコマンドを叩いてバージョン確認します。

```bash
% uloop --version
1.3.0
```

## Skillsのインストール

最後に、Claude Codeから呼び出せるSkill（スラッシュコマンド）をインストールします。

Settingsの `Target` から「Claude」を選択。

![Target選択](https://storage.googleapis.com/zenn-user-upload/5265ba3589b6-20260324.png)

「Install Skills」をクリック。

![Install Skills](https://storage.googleapis.com/zenn-user-upload/2dcb43cd2a7c-20260324.png)

これでインストール完了です。

![Skills一覧](https://storage.googleapis.com/zenn-user-upload/0e7d1b80b202-20260324.png)

## バンドルされているSkill

インストールされるSkillは、Claude Codeで `/uloop-***` のようなスラッシュコマンドとして利用できます。以下のような自然言語の指示で対応するSkillが呼ばれます。

| 指示例 | 使用スキル |
|--------|-----------|
| Launch Unity for this project | `/uloop-launch` |
| Fix the compile errors | `/uloop-compile` |
| Run the tests and tell me why they failed | `/uloop-run-tests` + `/uloop-get-logs` |
| Check the scene hierarchy | `/uloop-get-hierarchy` |
| Search for prefabs | `/uloop-unity-search` |
| Play the game and bring Unity to the front | `/uloop-control-play-mode` + `/uloop-focus-window` |
| Bulk-update prefab parameters | `/uloop-execute-dynamic-code` |
| Take a screenshot of Game View and adjust the UI layout | `/uloop-screenshot` + `/uloop-execute-dynamic-code` |
| Record my gameplay input | `/uloop-record-input` |
| Replay the recorded input | `/uloop-replay-input` |

全20スキルがバンドルされており、`.claude/skills` 配下の `SKILL.md` で各スキルの詳細が確認できます。

- `/uloop-launch` - Unity起動
- `/uloop-compile` - コンパイル実行
- `/uloop-get-logs` - コンソールログ取得
- `/uloop-run-tests` - テスト実行
- `/uloop-clear-console` - コンソールクリア
- `/uloop-focus-window` - エディタをフォーカス
- `/uloop-get-hierarchy` - シーン階層取得
- `/uloop-unity-search` - Unity検索
- `/uloop-get-menu-items` - メニュー項目取得
- `/uloop-execute-menu-item` - メニュー実行
- `/uloop-find-game-objects` - GameObjects検索
- `/uloop-screenshot` - エディタウィンドウキャプチャ
- `/uloop-simulate-mouse-ui` - UIマウス操作シミュレーション
- `/uloop-simulate-mouse-input` - PlayModeマウス入力シミュレーション
- `/uloop-simulate-keyboard` - キーボード入力シミュレーション
- `/uloop-record-input` - 入力記録
- `/uloop-replay-input` - 入力再生
- `/uloop-control-play-mode` - PlayMode制御
- `/uloop-execute-dynamic-code` - 動的C#コード実行
- `/uloop-get-unity-search-providers` - 検索プロバイダー詳細取得

## 動作確認

Claude Codeに簡単な機能実装を指示し、以下の操作が動作することを確認できました。

- コンパイルチェック、コンソールログの取得
- Unity EditorのPlayモード起動
- スクリーンショット撮影
- ゲーム画面内のUIボタン座標確認とタップ操作
- ゲーム進行状況の把握、ステータス数値の変動確認

特にマウス操作のシミュレーション機能が便利そうです。AIエージェントにゲーム画面上のUI操作まで任せられるのは大きいですね。
