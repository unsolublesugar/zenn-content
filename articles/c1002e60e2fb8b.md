## Playwright MCPとは

Playwright MCPは、Microsoft製のMCPサーバーです。AIエージェントがブラウザを操作し、Webページのスクリーンショット撮影やナビゲーション、フォーム入力などを行えるようになります。

https://github.com/microsoft/playwright-mcp

## Playwright MCPの設定

`.devcontainer/.mcp.json` を作成し、以下の内容を記述します。

```json
{
    "mcpServers": {
        "playwright": {
            "command": "npx",
            "args": [
                "@playwright/mcp@latest",
                "--browser",
                "chromium",
                "--isolated",
                "--no-sandbox"
            ]
        }
    }
}
```

`.mcp.json`ファイル配置後にClaude Codeを起動すると、新規MCPサーバーの有効化確認が表示されます。

![MCPサーバー有効化確認](https://storage.googleapis.com/zenn-user-upload/268a0bb97f4e-20260309.png)

`1` を選択して有効化。

```
> 1. Use this and all future MCP servers in this project 
```

この設定は `/.devcontainer/.claude/settings.local.json` に保存されます。

```json
{
  "enabledMcpjsonServers": [
    "playwright"
  ],
  "enableAllProjectMcpServers": true
}
```

書籍ではサンプルのブラウザゲームのスクショを撮る形になっていましたが、一般的なWebサイトを開けるか見てみることにしました（E2Eテスト的なことができそうだったので）。

Yahoo! JAPANのトップページでスクリーンショット撮影を試してみます。

### ブラウザ未インストールエラー

初回実行時、Dev Container環境にはブラウザがインストールされていないためエラーが発生しました。

```shell
● playwright - Navigate to a URL (MCP)(url: "https://www.yahoo.co.jp")

     Error: Browser "chromium" is not installed. Either install it (likely) or change the config.
```

Claude Codeが自動的にChromiumをインストール。


## 書籍で紹介されていたMCPサーバー

書籍で紹介されていたMCPサーバーと、参考になりそうな記事をまとめておきます。

### Context7 MCP

AIが参照するドキュメントのコンテキストを調整し、最新または特定バージョンの情報をAIに提供するMCPサーバー。

https://github.com/upstash/context7

https://zenn.dev/spacemarket/articles/21c4df0b7795f9

### Serena MCP

コーディングエージェントのツールキット。関数やクラスの検索・リネーム、関連コード追跡など、シンボルを使った検索や編集に対応。

https://github.com/oraios/serena

https://zenn.dev/suwash/articles/serene_mcp_20250807

https://zenn.dev/contrea/articles/d18ee9447a9366

https://zenn.dev/sc30gsw/articles/ff81891959aaef

### Chrome DevTools MCP

AIエージェント向けのChrome DevToolsサーバー。

https://github.com/ChromeDevTools/chrome-devtools-mcp

https://developer.chrome.com/blog/chrome-devtools-mcp?hl=ja

https://zenn.dev/nexta_/articles/google-chrome-mcp-server

https://zenn.dev/iret/articles/4dd51a9d3a2e6e

### Figma MCP

FigmaのデザインデータをAIエージェントから参照できるMCPサーバー。デザインからコードへの変換などに活用できます。

https://help.figma.com/hc/ja/articles/32132100833559-Figma-MCPサーバーのガイド

https://zenn.dev/reiwatravel/articles/13828b9659798a

https://zenn.dev/kimmaru/articles/6c0f67f6003b94

### Sentry MCP

エラー監視プラットフォームSentryのMCPサーバー。AIエージェントからSentryのIssueやエラー情報を取得・分析できます。

https://github.com/getsentry/sentry-mcp
